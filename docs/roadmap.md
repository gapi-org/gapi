# Roadmap

Gapi is core-complete for an initial public alpha. The current release includes the typed handler workflow, request binding, validation, Problem Details, OpenAPI docs, middleware, dependency injection, response helpers, CLI, examples, and testing helpers.

Planned before v1:

- Dependency overrides for tests and larger applications.
- JWT and OAuth2 helpers.
- Recursive JSON Schema references and custom schema providers.
- Full typed client generation.
- Optional code generation for binders, validators, and schemas.
- Router adapters for chi, gin, and echo.
- CLI live reload for local development.
- More benchmarks, fuzz targets, and production examples.
